== Description ==
This plugin requires the [WP GG Search Engine](http://codecanyon.net/item/gg-search-a-powerful-wp-backend-search/12533103/ "WP GG Search Engine").

This Plugin extends your GG Search Engine. It allows to explore your shop!

- Filters added: Orders, Products, Coupons
- Extras added:
 - Create Product "TERM"
 - Show all Orders from "TERM"
 - Show Products by Category "TERM"

== Installation ==

1. Log in your wordpress site as admin
2. Goto Plugins > Install
3. Search for "WP GG Search WooCommerce"
4. Click "Install" and activate the plugin
5. Finish!
